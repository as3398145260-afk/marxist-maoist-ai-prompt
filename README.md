# 辩证实践引擎 (Dialectical Praxis Engine) - AI 思维重塑 Prompt

**当前版本**: v15.0

## 📖 项目简介

本项目是一次实验性的探索：**将马克思列宁主义与毛泽东思想的方法论（辩证唯物主义、历史唯物主义、实践论）编写为大语言模型（LLM）的底层思维协议（Prompt）。**

传统的 AI 往往停留在现象表层进行“端水大师”式的概率文本生成。本 Prompt 旨在重构 AI 的注意力机制和逻辑推演路径，强制 AI 在输出最终回答前，必须经历一个深度、自然、未经过滤的内心独白过程，用动态的、物质的、充满矛盾的眼光去解剖问题。

理论必须与实践相结合，本项目目前仍在不断迭代中。我们需要广泛的测试与实践案例来暴露现有框架的局限性，从而实现 Prompt 的螺旋式上升。

---

## ✨ 核心亮点 (Key Highlights)

1. **绝对的物质锚定 (Absolute Material Anchoring)**：彻底杜绝 AI 的“高空建瓴”和“废话文学”。强制 AI 追问：事物的矛盾是由什么具体的底层硬件、资金流、物理规则或生理极限驱动的？
2. **宏大理论的微观降维**：不仅能分析国际局势，更能用“生产力与生产关系”、“经济基础与上层建筑”的框架，精准降维打击个人的职场困境、情感问题或具体的代码架构设计。
3. **拒绝庸俗折中主义**：严禁 AI 给出“A有理，B也有理”的各打五十大板结论。强制要求寻找主要矛盾，给出清晰的历史演进方向。
4. **动态生命周期与新矛盾预判**：不仅给出当前问题的解决方案，还强制要求 AI 预判“在执行该方案后，会在什么新的物质载体上激发出什么新的矛盾”，真正做到走一步看三步。

## 🧠 思考流程解析 (The Thinking Flow)

当 AI 接收到你的问题后，它在后台的思考将严格遵循以下“辩证螺旋”：

1. **初见与祛魅 (Initial Engagement)**：剥离用户语言的表象，识别问题背后的真实目的和物质处境。
2. **提取本质 (Problem Analysis)**：锁定主要矛盾，寻找承载该矛盾的具体物理/社会实体（是服务器算力？是利益分配机制？还是时间资源？）。
3. **矛盾运动推演 (Materialist Contradiction Movement)**：使用“两点论”剖析事物的进步性与其隐性代价。推演当物质/资源消耗到什么“度”时，会发生强制性的“质变”。
4. **知识扬弃与综合 (Knowledge Synthesis)**：打破初步结论，向下深挖更底层的历史或经济动因，完成认识的二次深化。
5. **化为实践 (Actionable Synthesis)**：结束内心独白，输出对用户真正有用的战术指导和降维分析。

## 🎯 推荐使用场景 (Recommended Scenarios)

这个引擎在处理那些**表面繁杂、内藏利益博弈或资源瓶颈**的问题时，表现最为惊艳：

*   **⚡ 个人破局与职场决策**：遇到职业瓶颈、办公室政治或人际冲突时，让 AI 帮你剥离情绪，用“生产关系变动”的视角找出利益的根本锚点和破局方向。
*   **💻 行业洞察与技术趋势解析**：评估某项新技术（如 AI、Web3、新能源）到底是在解放生产力，还是在制造新的异化和垄断？谁掌握了新的生产资料？
*   **📰 社会热点与商业模式剖析**：透过公关话术和媒体报道的迷雾，用历史唯物主义的刻刀解剖商业模式背后真实的代价转移机制和阶级/群体利益分配。
*   **🤔 深度哲学与战略推演**：作为“同志式的诤友”，用来进行深度的头脑风暴，推演复杂项目的长期生命周期和潜在危机。

## 🚀 如何使用 (Usage)

为了获得最佳的“深度思考”效果，建议按以下方式操作：

### 方案一：使用 API 客户端（最佳实践 🌟）
这是能够最大程度激发该方法论效力的使用方式，适用于支持自定义系统提示词（System Prompt）的工具。
* **推荐工具**:[Cherry Studio](https://github.com/CherryHQ/cherry-studio), [Google AI Studio](https://aistudio.google.com/), 或 NextChat, LobeChat 等。
* **操作步骤**:
  1. 复制本项目中 `v15.md`（或最新版本）的全部内容。
  2. 将其粘贴到客户端的 **System Prompt（系统设定）** 输入框中。
  3. 像往常一样对话。
  

### 方案二：使用 DeepSeek 官方软件/网页版（文件外挂法）
由于 DeepSeek 官方对话框难以长久记住超长的系统指令，请采用此折中方案。
* **操作步骤**:
  1. 将本项目中的 Prompt 文本保存为一个本地的 `.md` 文件。
  2. 在 DeepSeek 的对话框中，**点击上传该 `.md` 文件**。
  3. 在同一个对话框中输入你的问题，并附加一句指令：“**请严格按照我上传的 .md 文件中的思维协议来分析并回答以下问题：[你的问题]**”。
  4. **成功标志**：如果 AI 顺利触发了其内置的深度思考模型，你会在界面上看到 `<think> ... </think>` 标签。只要它展开了长篇大论的思考，就说明方法论注入成功！反之，如果直接给出了干瘪的结论，请点击重新生成。

## 🛠️ 欢迎魔改与定制 (Customization is Encouraged!)

**“把灵魂注入你想要的躯壳。”**

这个 Prompt 的本质是一个**“强制深度思考的逻辑引擎”**。我们极度欢迎并且鼓励大家 Fork 这个项目，并根据自己的需求进行魔改！

*   **如果你不喜欢马列毛框架**：你可以把核心指导思想替换为“第一性原理 (First Principles)”、“斯多葛学派 (Stoicism)”、“系统动力学 (Systems Thinking)”甚至是“王阳明心学”。
*   **保留引擎核心**：在替换思想流派时，建议保留 `<core_thinking_sequence>`（思考流程）和 `<verification_quality_control>`（质量控制）的结构，因为这是保证大模型不胡说八道、不产出废话的物理外壳。
*   **打造你自己的专属智库**：调整 `<analysis_tier_control>`（分析层级）以适应你是做金融量化、写代码还是写小说的特殊场景。
